URL Parser

URL Parser computes the mathematical expression passed in the form of url.

JSON object is returned as a result which consists both the question and answer.

Sample Requests
GET Request Response
Endpoint / HTML
Homepage
Endpoint /history HTML
Lists the last 20 operations performed on the server and the answers.

How-To Guide

  1. Download & extract the zip file from this github link.
  2. Now, open terimal and run the following commands one by one:
    => npm i
    => npm start
  3. On successfully running the server locally, open the browser and type: /{mathematical expression as shown in sample requests}
  4. Example url: /18/plus/12/minus/12
  5. Response: {question: "18+12-12", answer: "18.00"}
  6. History of last 20 operations can be checked using the endpoint: /history
  7. List of Operators available: plus( + ), minus( - ), into( * ), by( / ), bitwise XOR( ^ ), bitwise OR( | ) and bitwise AND( & )
