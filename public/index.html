URL Parser

URL Parser computes the mathematical expression passed in the form of url.

JSON object is returned as a result which consists both the question and answer.

How-To Guide

  1. Download & extract the zip file from this github link.
  2. Now, open the terminal and run the index.js file by using the command: node index.js
  3. On successfully running the server locally, open the browser and type localhost:port/{mathematical expression as shown in sample requests}
  4. Example url: localhost:3000/18/plus/12/minus/12
  5. Response: {question: "18+12-12", answer: "18"}
  6. History of last 20 operations can be checked using the endpoint: localhost:3000/history
  7. List of Operators available: plus( + ), minus( - ), into( * ) and by( / )
